📘 语言概览

Mochi 3.4 是一款现代化动态脚本语言，拥有类与继承、原生并发 Actor 模型、丰富的数据结构、运算符重载以及强大的 FFI。本版本新增 扁平化 GUI 系统，支持窗口、按钮、文本框（含密码模式）、滑块、进度条、多行编辑器等控件，所有界面更新线程安全。关键字 loc 用于声明局部变量。

特性说明
类与单继承class / super / this / 动态方法 / 自动析构
变量声明loc name = value;
数据结构数组 [] / 字典 {} / 字符串索引与转义
函数与闭包fn 关键字，一等公民，环境捕获
控制流if / while / for / for-in / break / continue
运算符+ - * / % 及全部比较运算符，支持重载
错误处理throw / try-catch / error()
模块系统import("name") 内置 / "file.mochi" 脚本 / "lib.mochimod" 动态库
迭代器协议for-in 支持 range()、数组、字典、自定义 iter()
并发模型spawn / send / ask / reply / Future / close / join
运算符重载__add__, __mod__, __getitem__ 等特殊方法
深拷贝clone() 与 __clone__ 协议
FFI双向调用 C++ 函数与 Mochi 对象
GUI 系统窗口、按钮、文本框、滑块、进度条、多行编辑器等，线程安全
REPL交互式环境，支持多行与状态保持

🔢 类型与字面量

基础类型

  • nil – 空值
  • bool – true / false
  • number – 双精度浮点（如 3.14）
  • string – 双引号，支持转义（\n, \t, \\, \", \0）

复合类型

  • class – 自定义类
  • instance – 类的实例

🔄 控制流

// 条件
if (x > 10) {
    print("big");
} else {
    print("small");
}

// while 循环
loc i = 0;
while (i < 3) {
    print(i);
    i = i + 1;
}

// for 循环
for (loc j = 0; j < 5; j = j + 1) {
    if (j == 2) continue;
    if (j == 4) break;
    print(j);
}

// for-in + range
for (loc n in range(0, 5)) {
    print(n);   // 0 1 2 3 4
}

// 数组迭代
for (loc item in [10, 20, 30]) {
    print(item);
}

🧬 类与继承

class Animal {
    init(name) { this.name = name; }
    speak() { print(this.name + " makes a sound."); }
}

class Dog : Animal {
    init(name, breed) { super.init(name); this.breed = breed; }
    speak() { print(this.name + " barks!"); }
}

loc d = Dog("Buddy", "Golden");
d.speak();  // Buddy barks!

动态方法与自动析构：

d.wag_tail = fn() { print(this.name + " wags."); };
d.wag_tail();

class Resource {
    init(name) { this.name = name; print("Create " + name); }
    destroy() { print("Destroy " + this.name); }
}
{
    loc r = Resource("obj"); // 离开作用域自动调用 destroy
}

class Vector {
    init(x, y) { this.x = x; this.y = y; }
    __add__(v) { return Vector(this.x + v.x, this.y + v.y); }
    __str__()  { return "(" + this.x + ", " + this.y + ")"; }
}
loc v = Vector(1,2) + Vector(3,4);
print(v);  // (4, 6)

📦 模块系统

  • 内置模块：直接写名称，如 import("fs")

⚡ Actor 并发模型

loc worker = spawn(fn(msg) {
    reply("Result: " + msg);
});
loc future = worker.ask("query");
print(future.get());   // Result: query
worker.close(); worker.join();

🔌 FFI 外部函数接口

// C++ 侧注册
ffi->add("add", [](Interpreter*, auto& args) {
    return Value(std::any_cast(args[0]) + std::any_cast(args[1]));
});
// Mochi 侧
loc sum = ffi.call("add", 3, 4);   // 7

🖼️ GUI 图形界面（v3.4 新增）

基于 SDL2 的扁平化 GUI 库，使用 import("gui") 加载。所有界面操作入队主线程，可在 Actor 中安全调用。

基础控件

loc gui = import("gui");
loc win = gui.createWindow("My App", 400, 300);
loc layer = gui.addLayer(win, "main");

loc btn = gui.addButton(layer, "btn", "Click me", 100, 100, 120, 40, fn() {
    print("Clicked!");
});
gui.addLabel(layer, "title", "Hello GUI", 50, 30);
gui.run();

📝 示例：登录框


    loc gui = import("gui");

    loc win = gui.createWindow("布局系统演示",560, 480, 4);
    loc layer = gui.addLayer(win, "main");

    // 垂直布局（左侧面板）—— 高度充足，控件不会溢出
    loc vbox = gui.addVLayout(layer, "leftPanel", 15, 15, 180, 410);
    vbox.setSpacing(6);
    vbox.setPadding(10);

    vbox.addLabel("用户信息");
    vbox.addTextField("用户名", 150, 28);
    vbox.addTextField("密码", 150, 28);
    vbox.addCheckBox("自动登录", 140);
    vbox.addButton("登录", 140, 32);
    vbox.addSlider(0, 100, 1, 150);

    // 水平工具栏
    loc hbox = gui.addHLayout(layer, "topBar", 210, 15, 320, 40);
    hbox.setSpacing(10);
    hbox.addLabel("编辑工具");
    hbox.addButton("剪切", 65, 28);
    hbox.addButton("复制", 65, 28);
    hbox.addButton("粘贴", 65, 28);

    // 多行编辑区
    loc memo = gui.addMemo(layer, "editor", 210, 70, 320, 310);
    gui.setMemoText(memo, "这是 Mochi GUI 布局示例。\n你可以在这里编辑文本。");

    // 底部按钮
    gui.addButton(layer, "saveBtn", "保存内容", 15, 440, 120, 32, fn() {
    loc text = gui.getMemoText(memo);
    print("保存的文本: " + text);
    });
    gui.addButton(layer, "quitBtn", "退出", 430, 440, 120, 32, fn() {
    gui.quit();
    });

    gui.run();

所有 GUI 操作均通过 import("gui") 模块完成。控件采用搭积木风格，布局和样式自动处理。详细请看mochi_gui.html

🧩 第三方动态库（.mochimod）

使用 C++ 编写模块并编译为动态库，Mochi 可通过 import 加载。需实现 extern "C" Value mochi_module_init(Interpreter*)。提供 mochi_api.h 头文件。

// 编译：g++ -std=c++17 -fPIC -shared hello_module.cpp -o hello.mochimod
// Mochi：loc hello = import("hello.mochimod");
