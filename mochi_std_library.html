📁 fs — 文件系统

函数列表

函数参数说明
fs.read(path)path：文件路径（字符串）读取文件全部内容，返回字符串
fs.write(path, content)path, content（字符串）将内容写入文件（覆盖）
fs.append(path, content)path, content追加内容到文件末尾
fs.exists(path)path检查文件或目录是否存在，返回布尔
fs.remove(path)path删除文件（或空目录）
fs.listdir(path)path（可选，默认当前目录）列出目录下的文件和子目录名，返回数组

示例

loc fs = import("fs");
fs.write("test.txt", "Hello Mochi!");
print(fs.read("test.txt"));  // Hello Mochi!
fs.append("test.txt", "\n新的一行");
print(fs.listdir("."));      // ["test.txt", ...]
fs.remove("test.txt");

常量

📦 json — JSON 处理

函数列表

函数参数说明
json.encode(value)任意 Mochi 值将值序列化为 JSON 字符串
json.decode(str)JSON 字符串解析 JSON 字符串为 Mochi 值

示例

loc json = import("json");
loc obj = {"name":"Mochi","age":3.4};
loc s = json.encode(obj);        // {"name":"Mochi","age":3.4}
loc copy = json.decode(s);
print(copy.name);                // Mochi
🎲 random — 随机数工具

函数列表

函数参数说明
random.seed(n?)种子（数字，可选）设置随机种子，无参数时使用硬件随机
random.random()无返回 [0,1) 浮点数
random.uniform(a,b)下限a,上限b返回 [a,b) 浮点数
random.randint(min,max)两个整数返回 [min,max] 整数
random.choice(array)数组随机返回一个元素
random.shuffle(array)数组（原地）打乱数组顺序
random.sample(array,k)数组,数量k无重复抽取k个元素，返回新数组
random.bytes(n)字节数返回n个随机字节的字符串

示例

loc rnd = import("random");
rnd.seed(123);
print(rnd.random());        // 0.xxx
print(rnd.randint(1,6));    // 1~6
loc arr = [1,2,3,4,5];
rnd.shuffle(arr);
print(arr);
🔤 string — 字符串增强工具

函数列表

函数参数说明
string.split(str, delim)字符串，分隔符分割字符串，返回数组
string.join(array, delim)数组，分隔符用分隔符连接数组元素，返回字符串
string.trim(str)字符串去除首尾空白字符
string.startsWith(str, prefix)字符串，前缀判断是否以指定前缀开始
string.endsWith(str, suffix)字符串，后缀判断是否以指定后缀结束
string.upper(str)字符串转为大写
string.lower(str)字符串转为小写
string.replace(str, from, to)字符串,查找内容,替换内容替换所有匹配项
string.repeat(str, n)字符串,次数重复字符串n次

示例

loc s = import("string");
loc parts = s.split("a,b,c", ",");
print(parts);                    // ["a","b","c"]
print(s.join(parts, "-"));       // "a-b-c"
print(s.trim("  hello "));       // "hello"
print(s.repeat("ha", 3));        // "hahaha"
📊 array — 数组高级操作

函数列表

函数参数说明
array.map(arr, fn)数组,回调函数对每个元素调用fn，返回新数组
array.filter(arr, fn)数组,回调函数返回满足条件的元素组成的新数组
array.filterInPlace(arr, fn)数组,回调函数原地移除不满足条件的元素
array.reduce(arr, fn, init?)数组,累积函数,初值(可选)累积计算
array.find(arr, fn)数组,回调返回第一个满足条件的元素，或nil
array.every(arr, fn)数组,回调是否所有元素都满足条件
array.some(arr, fn)数组,回调是否存在满足条件的元素
array.sort(arr, cmp?)数组,比较函数(可选)原地排序，默认按字符串
array.reverse(arr)数组原地反转
array.copy(arr)数组浅拷贝数组
array.flatten(arr)数组（含嵌套）展平一层嵌套

示例

loc a = import("array");
loc nums = [1,2,3,4,5];
loc doubled = a.map(nums, fn(x) { return x*2; });
loc evens = a.filter(nums, fn(x) { return x%2==0; });
a.sort(nums, fn(x,y) { return y-x; });  // 降序
print(nums);

🌐 utf — Unicode 辅助

函数列表

函数参数说明
utf.len(str)字符串返回 Unicode 字符数（非字节数）
utf.sub(str, start, count?)字符串,起始索引,数量(可选)按字符截取子串
utf.charAt(str, index)字符串,索引返回指定索引的字符（字符串）
utf.codes(str)字符串返回 Unicode 码点数组

示例

loc u = import("utf");
loc s = "Hello世界";
print(u.len(s));        // 7
print(u.sub(s,5,2));    // "世界"
print(u.charAt(s,5));   // "世"
print(u.codes("A世"));  // [65, 19990]
📂 dir — 目录操作

函数列表

函数参数说明
dir.mkdir(path)路径创建目录
dir.rmdir(path)路径删除空目录（失败抛出异常）
dir.isdir(path)路径判断是否是目录
dir.cwd()无获取当前工作目录
dir.chdir(path)路径改变当前工作目录
dir.walk(path)路径递归遍历目录，返回可迭代对象（每个元素为字典含 path,name,isdir）

示例

loc d = import("dir");
print(d.cwd());
d.mkdir("test");
d.chdir("test");
for (loc item in d.walk(".")) { print(item.name); }
d.chdir("..");
d.rmdir("test");
🌐 socket — 网络通信 (v3.4)

函数列表

函数参数说明
socket.tcp()无创建 TCP 套接字代理
socket.udp()无创建 UDP 套接字代理
socket.bind(sock, addr, port)套接字,地址字符串,端口号绑定本地地址
socket.listen(sock, backlog)套接字,等待队列长度开始监听（仅TCP）
socket.accept(sock)套接字接受客户端连接，返回新套接字代理
socket.connect(sock, addr, port)套接字,地址,端口连接到远程服务器
socket.send(sock, data)套接字,字符串发送数据，返回发送字节数
socket.recv(sock, maxSize)套接字,最大字节数接收数据（阻塞），连接关闭返回空串
socket.close(sock)套接字关闭套接字
socket.select(sockArray, timeout?)套接字数组,超时秒(可选,-1永久)返回可读的套接字数组
socket.handle(sock)套接字返回底层套接字句柄（数字）

示例（回显服务器）

loc sock = import("socket");
loc server = sock.tcp();
sock.bind(server, "0.0.0.0", 8080);
sock.listen(server, 5);
loc sockets = [server];
loc srvH = sock.handle(server);
while (true) {
    loc ready = sock.select(sockets, 0.5);
    for (loc s in ready) {
        if (sock.handle(s) == srvH) {
            loc cli = sock.accept(server);
            sockets.push(cli);
        } else {
            loc data = sock.recv(s, 1024);
            if (data == "") { sock.close(s); ... }
            else { sock.send(s, "Echo: "+data); }
        }
    }
}
🧰 pack — 二进制打包与位运算 (v3.4)

打包/解包

函数参数说明
pack.pack(fmt, args...)格式字符串, 值列表按格式将值打包为二进制字符串
pack.unpack(fmt, data)格式字符串, 二进制字符串解包二进制数据，返回数组

位运算

pack.bitand(a,b)两个数字按位与
pack.bitor(a,b)两个数字按位或
pack.bitnot(a)数字按位非
pack.bitxor(a,b)两个数字按位异或
pack.lshift(a,n)数字,位数左移
pack.rshift(a,n)数字,位数逻辑右移
pack.arshift(a,n)数字,位数算术右移（保留符号）

进制转换

pack.hex(str)十六进制字符串转为数字
pack.oct(str)八进制字符串转为数字
pack.bin(str)二进制字符串转为数字

格式字符

字节序前缀：@ = < > !，默认 >（大端）。常用字符：b B h H i I q Q f d s（需带数字如4s）c。

示例

loc pk = import("pack");
loc bin = pk.pack(">i4s", 1024, "hello");
loc vals = pk.unpack(">i4s", bin);
print(pk.bitand(pk.hex("FF"), pk.hex("0F"))); // 15
print(pk.lshift(1, 3));   // 8
print(pk.bin("1101"));    // 13

💬 dialog — 系统对话框 (v3.4)

函数列表

函数参数说明
dialog.info(title, msg)标题,消息显示信息框
dialog.warning(title, msg)标题,消息显示警告框
dialog.error(title, msg)标题,消息显示错误框
dialog.question(title, msg)标题,消息询问是/否，返回布尔
dialog.input(title, msg, def?)标题,提示信息,默认值(可选)输入框，返回字符串或nil
dialog.openFile(title, filters?, multi?)标题,过滤器字符串,是否多选打开文件对话框，返回路径字符串或多选路径数组
dialog.saveFile(title, filters?)标题,过滤器保存文件对话框，返回路径
dialog.selectFolder(title, defPath?)标题,默认路径选择文件夹，返回路径

示例

loc dlg = import("dialog");
dlg.info("提示", "操作完成！");
loc yes = dlg.question("确认", "是否继续？");
if (yes) {
    loc name = dlg.input("用户名", "请输入:");
    if (name) print("你好，" + name);
}
loc file = dlg.openFile("选择文本", "*.txt");
if (file) print(import("fs").read(file));
